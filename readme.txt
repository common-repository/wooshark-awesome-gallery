== Description ==

Import products from Wish to your woocommerce store in 1 click

FEATURES:

Wooshark dropshipping for Wish offer the following set of features

Import products from Wish to woocommerce including

1- image
2- Description
3- Title
4- Price
5- Quantity
6- sku

Search products by keywords
check the list of imported products
link to oringinal product




MINIMUM REQUIREMENTS
WordPress 4.7+
WooCommerce 3.0.0+
PHP version 5.6+
